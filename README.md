# Kuri Labo

Kuri Labo is an iOS-first mobile app for nail salon booking, customer management, and nail inspiration browsing.

The app is designed with a cute, soft, minimal aesthetic inspired by Japanese and Korean nail salon apps.

---

# Features

## User Features

- Register and login
- Manage personal profile
- View personal nail collection history
- Browse nail inspiration images
- Book available appointment slots

## Admin Features

- Manage users
- Manage appointments
- Create available booking slots
- Upload nail images for users
- Upload inspiration images

# Design Philosophy

The app should feel:
- soft
- clean
- cozy
- visual-first
- elegant but cute

Avoid:
- enterprise dashboard feeling
- overly complex UI
- heavy text-based layouts
- dark tech aesthetic
